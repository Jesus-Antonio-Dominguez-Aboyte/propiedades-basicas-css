Propiedades Basicas CSS / herencia

Esta practica muestra las propiedades basicas el CSS, por cada contenedor hijo puedes aplicar estilos al contenedor padre

dentro de este parrafo y practica hay diferentes propiedades Basicas de las cuales tenemos las siguientes:

  • background-color: color al fondo.
  • color: color al texto.
  • font-size: tamaño de la fuente.
  • font-family: tipo de fuente.
  • font-style: estilo de fuente ya sea italia o cursiva.
  • text-align: alineacion del texto.
  • padding: espacio alrededor del texto.
  • margin:espacio fuera del borde.
  • border: borde alrededor de la caja.

    • estas son algunas de las que se usan en esta practica.

    Especificidad

    la especificidad es el peso que tienen los selectores en css

    1. selectores de tipo (p, div, h1,etc) vale 1 punto
    2. selectores de clase, atributo y pseudo clase vale 10 puntos
    3. selecroes de id son mas especipicos que los anteriores valen 100 puntos
    4. por ultimo los selectores de !important valen 1000 puntos estos deben se ocupados con cuidado

    FUUENTES

    EXISTEN DIFERENTES TIPOS DE FUENTES DESDE LAS PRETERMINADAS A LAS FUENTES WEB

    1. Serif: tiene pequeñas lineas al final del trazo
    2. san-serif: no tiene los adornos de serifas
    3. monospace: todas las letras tienen el mismo ancho
    4. cursive: tiene letras curcivas imitando una mano
    5. fantasy: fuentes mas decoraticas

    propiedades para las fuentes

    • font-size: pequeña fuente.
    • font-weight: peso de la fuente
    • font-style: estilo de fuente ya sea italia o cursiva.
    • line-height: altura de la fuente
    • letter-spacing: espacio entre las letras
    • text-transform: mayusculas o minusculas.
